# project2
Fitness Website

A responsive Fitness / Gym website built using HTML, CSS, and JavaScript.
This project provides information about gym facilities, workout programs, and tools like a BMI calculator to help users track their fitness.

📌 Project Overview

The Fitness Website is designed to promote a healthy lifestyle by providing users with easy access to workout programs, gym facilities, and fitness information.
It features a modern design, responsive layout, and interactive elements to enhance the user experience.
